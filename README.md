📌 Descripción

Este proyecto web explora la evolución de la arquitectura italiana a través de los siglos mediante una experiencia interactiva y educativa. Combina historia, cultura y tecnología para facilitar el aprendizaje y la exploración de hitos arquitectónicos.

✨ Características principales

- Línea de tiempo dinámica con carrusel de imágenes que muestra construcciones desde la Roma antigua hasta la arquitectura contemporánea.

- Mapa cultural interactivo con información y datos curiosos sobre ciudades como Roma, Florencia, Milán y Venecia.

- Sistema bilingüe (inglés/español) que adapta la experiencia a distintos públicos.

- Diseño accesible y adaptable, siguiendo buenas prácticas de usabilidad y responsive design.

- Interactividad mejorada mediante menús desplegables, animaciones y navegación fluida.

🛠️ Decisiones técnicas

Modulación en JavaScript:
- El código se organizó en módulos (translate.js, timeline.js, etc.) para mejorar la mantenibilidad y permitir funciones específicas como setLanguage.

Accesibilidad:
- Se implementaron etiquetas semánticas en HTML, contraste adecuado en CSS y soporte para traducción dinámica.

Interactividad:
- Uso de eventos en JavaScript para menús, carruseles y actualización de contenido en tiempo real.

Gestión de versiones con Git/GitHub:
- El proyecto se versiona con Git y se publica en GitHub Pages para facilitar la colaboración y el despliegue.

🎯 Objetivo

Ofrecer una experiencia educativa y visualmente atractiva que combine historia, cultura y tecnología, fortaleciendo la comprensión de la arquitectura italiana a través de herramientas digitales modernas.

El proyecto está disponible en GitHub Pages: https://pb-michelle.github.io/italy-architecture-website/
