# italy-architecture-website
Living Architecture: Italy es un proyecto web interactivo que explora la evolución de la arquitectura italiana a través de los siglos. Incluye:

 - Una línea de tiempo dinámica con carrusel de imágenes que muestra hitos arquitectónicos desde la Roma antigua hasta la arquitectura contemporánea.

 - Un mapa cultural interactivo con información y datos curiosos sobre las principales construcciones en ciudades como Roma, Florencia, Milán, Venecia y más.

 - Un sistema de traducción bilingüe (inglés/español) que permite adaptar la experiencia a distintos públicos.

 - Implementación de JavaScript, HTML y CSS para crear menús desplegables, elementos interactivos y una navegación fluida.

El objetivo del proyecto es ofrecer una experiencia educativa y visualmente atractiva, combinando historia, cultura y tecnología para facilitar el aprendizaje y la exploración de la arquitectura italiana.
